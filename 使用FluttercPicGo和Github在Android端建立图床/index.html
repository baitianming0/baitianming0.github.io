使用Flutter-PicGo和Github在Android端建立免费图床

Posted on 2022-01-02

使用FluttercPicGo和Github在Android端建立图床

简介:既然来看教程肯定是知道图床是干什么的~o(〃’▽’〃)o

Github

  1. 注册Github账号
    • 进入Github
    • 点击Sign up
    • 按照提示填入相关信息即可
  2. 建立仓库
    • 回到主页
    • 点击Start a Project
    • Repository name填入仓库名即可
    • 点击Create repository
    • 完成
  3. 获取Token
    • 点击左上角三横
    • 点击Settings
    • 点击Developer Settings
    • 点击Personal access tokens
    • 点击Generate new token
    • Note任意填入
    • Expiration选择No Expiration
    • 勾选repo
    • 点击Generate token
    • 记下Token
    • 完成

      PicGo

  4. 安装PicGo
    • 打开下载地址
    • 选择最新版本
    • 点击Assets
    • 选择app-release.apk
    • 下载完成后安装
    • 完成
  5. 设置
    • 打开PicGo
    • 选择设置
    • 选择图床设置
    • 选择Github图床
    • 按提示填入相关信息
    • 选择保存
    • 选择设置为默认图床
    • 完成

      使用

  6. 上传图片
    • 回到相册
    • 点击右下角的悬浮按钮
    • 选择图片
    • 确定
    • 刷新
    • 完成
  7. 查看图片信息
    • 点击图片
    • 长按即可
  8. 获取图片链接
    • 在图片信息中点击图片链接即可自动复制图片链接至剪贴板
