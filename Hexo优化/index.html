Hexo优化

Posted on 2022-01-03

更换主题

  1. 输入cd Blogs
  2. 输入git clone https://github.com/miiiku/hexo-theme-flexblock.git ./themes/flex-block
  3. 输入vim _config.yml
  4. 修改themes:主题名

添加标签

  1. 输入cd Blogs
  2. 输入hexo new page 标签名
  3. 输入cd themes/主题名
  4. 输入vim _config.yml
  5. 在memu下添加标签标签名: /标签
  6. 输入cd
  7. 输入cd Blogs/source/标签
  8. 输入vim index.md
  9. 在data下添加type: "标签名"
  10. 保存退出上传

添加搜索

  1. algolia
    • 打开algolia
    • 按提示注册账号
    • 成功后回到主页
    • 点击侧边栏第二个按钮
    • 点击Create Index
    • Index name任意填写，但要记住
    • 确认后刷新网页
    • 点击侧边栏最后一个齿轮型按钮
    • 点击APi keys
    • 点击 All API Keys
    • 点击New API Key
    • Indices 填入刚刚填写的Index name，ACL选择search addObject deleteObject deleteIndex
    • 确认
    • 复制All API Key中刚刚生成的秘钥，则获得API Key
      按第一个按钮复制
    • 回到Your API Key
    • 复制Application ID
