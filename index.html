Hexo优化

更换主题

  1. 输入cd Blogs
  2. 输入git clone https://github.com/miiiku/hexo-theme-flexblock.git ./themes/flex-block
  3. 输入vim _config.yml
  4. 修改themes:主题名

添加标签

  1. 输入cd Blogs
  2. 输入hexo new page 标签名
  3. 输入cd themes/主题名
  4. 输入vim _config.yml
  5. 在memu下添加标签标签名: /标签
  6. 输入cd
  7. 输入cd Blogs/source/标签
  8. 输入vim index.md
  9. 在data下添加type: "标签名"
  10. 保存退出上传

使用FFmpeg和Github和Termux在Android端制作免费视频床

Posted on 2022-01-02

使用FFmpeg和Github和Termux制作视频床

注:1.选择(y/n)都选y
2.文件名指你的文件的名字，可自定义

Termux

  1. 下载Termux
    • 进入下载地址
    • 选择最新版本下载
    • 安装
  2. 配置
    • 打开Termux
    • 输入pkg update
    • 完成
    • 输入pkg install ffmpeg
    • 完成
    • 输入termux-setup-storage
    • 同意
    • 完成
  3. 切片
    • 系统新建文件夹
    • 把要切片的视频移动到新建的文件夹内
    • 打开Termux
    • 输入cd /storage/emulated/0/新建的文件夹名
    • 输入 ffmpeg -y -i 文件名.mp4 -vcodec copy -acodec copy -vbsf h264_mp4toannexb 文件名.ts
    • 获得[文件.ts]
    • 输入ffmpeg -i 文件名.ts -c copy -map 0 -f segment -segment_list 文件名.m3u8 -segment_time 5 文件名%03d.ts
    • 打开文件管理软件
    • 进入新建的文件夹
    • 把切好的视频移动到同一个新的文件夹(不包括原文件和第一个命令生成的[文件.ts])
    • 完成

      Github

  4. 注册Github账号
    • 进入Github
    • 点击Sign up
    • 按照提示填入相关信息即可
  5. 建立仓库
    • 回到主页
    • 点击Start a Project
    • Repository name填入仓库名即可
    • 点击Create repository
    • 完成
  6. 在Github仓库中创建文件夹
    • 点击creating a new file/创建新文件
    • 在文件名中输入任意文件名和/
    • 建立文件夹
    • 输入任意文件名
    • 点击commit new file
    • 点击upload file
    • 选择choose your file
    • 进入放切片视频的文件夹
    • 选择所有文件
    • 点击确定
    • 等待进度条消失
    • 点击Commit changes
    • 进入Github中仓库的文件夹
    • 删除一开始为了建立文件夹而建立的文件(不删除也可以，但是我有强迫症╮(‵▽′)╭)
    • 点击Delete file
    • 点击 Commit changes
    • 完成

      播放视频

  7. 打开浏览器
  8. 输入https://cdn.jsdelivr.net/gh/Github用户名/仓库名/Github仓库中的文件夹名/文件名.m3u8

大功告成Ｏ(≧▽≦)Ｏ

使用Flutter-PicGo和Github在Android端建立免费图床

Posted on 2022-01-02

使用FluttercPicGo和Github在Android端建立图床

简介:既然来看教程肯定是知道图床是干什么的~o(〃’▽’〃)o

Github

  1. 注册Github账号
    • 进入Github
    • 点击Sign up
    • 按照提示填入相关信息即可
  2. 建立仓库
    • 回到主页
    • 点击Start a Project
    • Repository name填入仓库名即可
    • 点击Create repository
    • 完成
  3. 获取Token
    • 点击左上角三横
    • 点击Settings
    • 点击Developer Settings
    • 点击Personal access tokens
    • 点击Generate new token
    • Note任意填入
    • Expiration选择No Expiration
    • 勾选repo
    • 点击Generate token
    • 记下Token
    • 完成

      PicGo

  4. 安装PicGo
    • 打开下载地址
    • 选择最新版本
    • 点击Assets
    • 选择app-release.apk
    • 下载完成后安装
    • 完成
  5. 设置
    • 打开PicGo
    • 选择设置
    • 选择图床设置
    • 选择Github图床
    • 按提示填入相关信息
    • 选择保存
    • 选择设置为默认图床
    • 完成

      使用

  6. 上传图片
    • 回到相册
    • 点击右下角的悬浮按钮
    • 选择图片
    • 确定
    • 刷新
    • 完成
  7. 查看图片信息
    • 点击图片
    • 长按即可
  8. 获取图片链接
    • 在图片信息中点击图片链接即可自动复制图片链接至剪贴板

使用Termux和Hexo和Github在Android端搭建免费个人网站

Posted on 2022-01-02 Edited on 2022-01-03

使用Termux和Hexo和Github搭建免费个人网站

简介:一直希望有一个个人网站，这次终于成功了，但是在路上遇到了好多挫折，因此，我写了这篇教程帮助大家更简单的搭建网站

注:1. Termux选择(y/n) 都选y
2. 文件夹名或文件名都是你的文件名，可以自定义
3. 用户名指自己的账号名
4. Termux完成时会出现一行~ $

Github

  1. 注册Github账号
    • 进入Github
    • 点击Sign up
    • 按照提示填入相关信息即可
  2. 建立仓库
    • 回到主页
    • 点击Start a Project
    • Repository name填入用户名.github.io即可
    • 点击Create repository
    • 完成
  3. 选择主题
    • 进入Settings，选择Pages
    • 点击Choose a themes
    • 点击Select theme
    • 点击Commit changes
    • 完成

      Termux

  4. 下载Termux
    • 进入下载地址
    • 选择最新版本下载
    • 安装
  5. 配置
    • 打开Termux
    • 输入pkg update
    • 完成
    • 输入pkg install git
    • 完成
    • 输入pkg install nodejs
    • 完成
    • 输入pkg install openssh
    • 完成
    • 输入pkg install vim
    • 完成
    • 输入npm install -g hexo-cli
    • 完成
    • 输入hexo init 文件夹名
    • 完成
    • 输入cd 文件夹名
    • 输入npm install hexo-deployer-git --save
    • 完成
    • 输入vim _config.yml
    • 输入i
    • 找到文末
    • 把文末替换为
    • 点击软键盘上的ESE
    • 输入:wq
    • 按一下手机自带输入法键盘右下角的确认键
    • 完成
  6. 获取秘钥
    • 输入git config --global user.name "用户名"
    • 输入git config --global user.email "注册Github时使用的邮箱"
    • 输入ssh-keygen -t rsa -C "注册Github时使用的邮箱
    • 出现下图时按确认，连按4次
    • 输入cat ~/.ssh/id_rsa.pub
    • 复制出现的那串字符
    • 把这串字符保存到便签备用
    • 完成

连接Github与本地

  1. 连接
    • 打开浏览器
    • 进入Github
    • 点击右上角三横
    • 点击Settings
    • 点击SSH and GPG keys
    • 点击New SSH Key
    • 填写信息
    • 点击Add SSH Key
    • 完成
  2. 上传文件
    • 打开Termux
    • 输入cd Blogs
    • 输入hexo g -d
    • 完成

打开https://用户名.github.io就可以发现你的个人网站已经完成
