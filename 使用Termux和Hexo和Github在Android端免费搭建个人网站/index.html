使用Termux和Hexo和Github在Android端搭建免费个人网站

Posted on 2022-01-02 Edited on 2022-01-03

使用Termux和Hexo和Github搭建免费个人网站

简介:一直希望有一个个人网站，这次终于成功了，但是在路上遇到了好多挫折，因此，我写了这篇教程帮助大家更简单的搭建网站

注:1. Termux选择(y/n) 都选y
2. 文件夹名或文件名都是你的文件名，可以自定义
3. 用户名指自己的账号名
4. Termux完成时会出现一行~ $

Github

  1. 注册Github账号
    • 进入Github
    • 点击Sign up
    • 按照提示填入相关信息即可
  2. 建立仓库
    • 回到主页
    • 点击Start a Project
    • Repository name填入用户名.github.io即可
    • 点击Create repository
    • 完成
  3. 选择主题
    • 进入Settings，选择Pages
    • 点击Choose a themes
    • 点击Select theme
    • 点击Commit changes
    • 完成

      Termux

  4. 下载Termux
    • 进入下载地址
    • 选择最新版本下载
    • 安装
  5. 配置
    • 打开Termux
    • 输入pkg update
    • 完成
    • 输入pkg install git
    • 完成
    • 输入pkg install nodejs
    • 完成
    • 输入pkg install openssh
    • 完成
    • 输入pkg install vim
    • 完成
    • 输入npm install -g hexo-cli
    • 完成
    • 输入hexo init 文件夹名
    • 完成
    • 输入cd 文件夹名
    • 输入npm install hexo-deployer-git --save
    • 完成
    • 输入vim _config.yml
    • 输入i
    • 找到文末
    • 把文末替换为
    • 点击软键盘上的ESE
    • 输入:wq
    • 按一下手机自带输入法键盘右下角的确认键
    • 完成
  6. 获取秘钥
    • 输入git config --global user.name "用户名"
    • 输入git config --global user.email "注册Github时使用的邮箱"
    • 输入ssh-keygen -t rsa -C "注册Github时使用的邮箱
    • 出现下图时按确认，连按4次
    • 输入cat ~/.ssh/id_rsa.pub
    • 复制出现的那串字符
    • 把这串字符保存到便签备用
    • 完成

连接Github与本地

  1. 连接
    • 打开浏览器
    • 进入Github
    • 点击右上角三横
    • 点击Settings
    • 点击SSH and GPG keys
    • 点击New SSH Key
    • 填写信息
    • 点击Add SSH Key
    • 完成
  2. 上传文件
    • 打开Termux
    • 输入cd Blogs
    • 输入hexo g -d
    • 完成

打开https://用户名.github.io就可以发现你的个人网站已经完成
